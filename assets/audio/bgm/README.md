# BGM 素材目录

把你的音乐文件放到本目录，**保持文件名不变**即可覆盖默认轨：

| 文件名 | 用途 | 默认名（主页切换显示） |
|--------|------|------------------------|
| `01_menu.wav` | 主页 | 主页轻律 |
| `02_explore.wav` | 地图探索 | 探索漫步 |
| `03_battle.wav` | 普通战斗 | 战斗脉冲 |
| `04_boss.wav` | Boss / 二阶段 | Boss压迫 |
| `05_victory.wav` | 胜利结算 | 胜利余韵 |

## 格式建议
- WAV / OGG 均可（若用 OGG，改 `AudioManager.BGM_TRACKS` 里对应 `file` 扩展名）
- 建议循环友好的 30s–2min 片段
- 音量不要过大，游戏内还有「音乐音量」滑条

## 阶段自动切换
- 主页 → `menu`
- 地图探索 → `explore`
- 普通战斗 → `battle`
- Boss / Boss 二阶段 → `boss`
- 胜利 → `victory`

主页左侧 **「♪ BGM：…」** 可循环试听全部五轨。
