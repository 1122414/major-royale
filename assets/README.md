# Assets 资源目录约定

```
assets/
├── fonts/          # UI 字体（开源中文字体）
├── audio/          # 音效 / BGM（可先空，程序占位音效仍可用）
└── sprites/
    ├── ui/         # HUD 图标、边框装饰
    ├── chars/      # 玩家 / 敌人像素立绘
    ├── cards/      # 卡面 icon
    └── bg/         # 菜单 / 探索 / 战斗场景底图
```

色板见 `src/ui/ui_colors.gd`。当前字体：LXGW WenKai（SIL OFL 1.1）。
