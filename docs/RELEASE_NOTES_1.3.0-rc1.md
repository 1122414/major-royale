# 《专业大逃杀》1.3.0-rc1 候选版本说明

## 本版定位

1.3.0-rc1 在 1.2.0-rc1 的十战校园路线、五专业构筑与实时答辩基础上，补齐了玩家可长期积累、可选择装配且有明确上限的局外成长闭环。它仍是交付发行方继续接入 Steamworks 的候选构建，不代表已取得真实 AppID、通过 Valve 审核或完成外部设备验收。

## 局外成长

- 一局结束后按胜利场次、事件次数、挑战档位和是否通关结算永久金币；同一局只会结算一次。
- 6 个天赋购买后永久解锁，最多同时装配 2 个。
- 9 件装备分为工具、徽章、纪念品 3 个槽位，同槽可自由替换。
- 生存训练、心态建设、资源规划、校友网络共 4 条永久强化，每条最多 3 级，价格逐级提高。
- 天赋、装备和强化在开局时冻结为本局配置；玩家中途换装不会改写已经开始的一局。
- 永久加成受到槽位、等级和成本约束，不会无限叠加；四档挑战的解锁条件仍只由挑战通关决定。

## 存档与兼容

- 新增 `user://meta_progression.json` 独立档案，保存永久金币、已结算局、天赋、装备和强化等级。
- 局外档案采用临时文件写入、原子替换和上一版备份，与现有一局安全存档相互独立。
- 早期只含金币的局外档案会为新增字段补空值；旧一局存档缺少局外配置时会按零加成恢复。
- 每局拥有独立结算标识，快速重开相同专业和种子也不会发生奖励碰撞。

## 界面

- 主菜单新增“局外成长”入口和永久金币余额。
- 成长中枢按天赋、装备、永久强化三栏展示购买、装配、替换、卸下、等级与价格状态。
- 页面继续采用参考图确定的青黑像素面板、青色信息边框、金色选中态与夜景校园背景。

## 候选版限制

- 尚未绑定真实 Steamworks AppID、Depot、Steam 成就 API 或 Steam Cloud。
- Windows 包仍需在独立 Windows 设备从 Steam 私有分支完成安装与整局测试。
- macOS 包尚未完成 Developer ID 签名、公证和另一台 Mac 的 Gatekeeper 验证。
- 商店胶囊、实际玩法预告片、生成素材披露与 Valve 审核不属于本地代码可替代的步骤。
