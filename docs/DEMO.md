# 演示录制说明（1.1.0-vertical-slice）

仓库已包含一段 43 秒、1920×1080、30 FPS 的无声竖切预览：`docs/demo-1.1.0-vertical-slice.mp4`。如需对外发布带解说的完整演示，建议录制 90–120 秒，并按四张参考图对应画面组织：

1. 夜景主菜单 → 开始游戏 → 横向展示五专业，选择「计算机」；
2. 白天校园自由移动，展示压力小地图、任务栏、五建筑与 `E` 交互；
3. 教学楼普通战：打出攻击、防御和技能牌，展示伤害、护盾、状态与结束回合；
4. 图书馆 AI 精英战：展示敌人档案、白名单行动、策略气泡和压力区；
5. 胜利后选择卡牌 / 属性 / 遗物奖励并返回校园；
6. 操场终局展示就业压力 Boss、唯一上岸者结算、成就与本局总结。

完整录屏基准为 1920×1080、60 FPS；可使用 macOS `Cmd+Shift+5` 录制，并在发布前加入配乐、操作声与解说。

录制时 AI 服务可关闭；本地兜底会保留相同界面和可玩流程。避免录入编辑器、终端、密钥或系统通知。
