# 《专业大逃杀》1.2.0-rc1 候选版本说明

## 本版定位

1.2.0-rc1 是首个覆盖完整十战校园路线、五专业构筑、实时答辩窗口与终极答辩 Boss 的 Steam 候选版本。它不是已通过 Valve 审核的公开构建；Steamworks AppID、私有分支安装、签名公证和商店素材仍需发行方完成。

## 核心玩法

- 五个专业各有三条构筑方向、独立主动/被动技能和专业遗物。
- 108 张卡牌、15 件遗物、14 个校园事件、10 名敌人组成一局十战路线。
- 敌方攻击与控制意图会打开实时答辩窗口；精准反驳、安全换位和正面招架会改变回合结算。
- 可输入数字种子复现牌序、事件、奖励、敌人意图与精英词缀。
- 四档挑战逐阶提高敌人生命、直接伤害与开场压力，并收紧动作窗口和校内恢复；通关只解锁下一档规则，不提供永久数值碾压。

## 稳定性与可访问性

- 版本化一局存档使用临时文件原子替换与上一版本备份。
- 在线 AI 默认关闭；离线白名单策略可完成全部 AI Native 战斗。
- 键鼠和手柄均可完成主流程，支持四档答辩时长、减少动态效果和震动开关。
- 16:9、16:10 与超宽屏采用等比安全区，避免拉伸或关键 UI 漂移。
- 菜单退出与桌面窗口关闭统一先停止播放器并释放音频流，再结束进程。

## 候选版限制

- 尚未绑定真实 Steamworks AppID、Depot、Steam 成就 API 或 Steam Cloud。
- Windows 包仍需在独立 Windows 设备从 Steam 私有分支完成安装与整局测试。
- macOS 包尚未完成 Developer ID 签名、公证和另一台 Mac 的 Gatekeeper 验证。
- 商店胶囊、实际玩法预告片、生成素材披露与 Valve 审核不属于本地代码可替代的步骤。
